## Setup

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Scripts

Run from project root.

| Script | Purpose |
|--------|---------|
| `python scripts/fetch_wikipedia_frequencies.py` | Fetch letter/bigram/trigram data → `data/frequencies/` |
| `python scripts/decrypt_task1.py` | Task 1: decrypt `data/task1/article_encrypted.txt`; writes `article_decrypted.txt`, `substitution_key.txt` |
| `python scripts/encrypt_task4_openssl.py` | Task 4: encrypt task4 files (OpenSSL); writes `data/task4/*.bin` |
| `python scripts/encrypt_task4.py` | Task 4: same, using PyCryptodome |
| `python scripts/task5_error_propagation.py` | Task 5: encrypt, corrupt 55th byte, decrypt; writes `data/task5/*.bin` |
| `python scripts/extract_task1_content.py` | Extract task1 report body into `docs/reports/coursework1/task1_content.tex` |
